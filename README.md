
# DotFiles
All the config files synced using stow.

## Dependencies
Install Stow and git
`pacman -S stow git`

## Clone the repo
`git clone https://github.com/goldr0g3r/dotfiles.git`

## Other Required Packages
`sudo pacman -S neovim eza bat zoxide fzf tmux`

## Steps to setup
 - `mv dotfiles .dotfiles`
 - `cd .dotfiles`
 - `stow .`
